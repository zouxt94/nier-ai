# Nier AI 版本记录

## v0.0.1 — 2026-05-17

### 已实现
- 暗色主题网站，全局禁用纯白色
- 5 页导航结构：首页 / 工具总览 / 模型对比 / 排行榜 / 免费试用
- 首页 Hero 区域，双层圆形渐变背景 + Nier AI 艺术字标题
- 搜索框
- 8 篇评测文章，卡片网格展示，支持加载更多
- 文章详情折叠展开阅读
- 模型对比功能：输入提示词 → 同时调用多个模型并排对比
- 排行榜：8 款主流模型参数/价格表，支持厂商筛选与排序
- 免费试用：选模型 → 选任务 → 输入内容 → 查看输出
- API Key 通过环境变量或 `.streamlit/secrets.toml` 配置，用户界面完全不可见
- 访问密码保护（可选，设环境变量 ACCESS_PASSWORD 开启）

### 技术栈
- Python + Streamlit
- OpenAI SDK（兼容 DeepSeek 等）
- 域名：nier-ai.com

- 文章搜索功能
